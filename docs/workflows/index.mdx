These guides describe complete outcomes rather than individual screens. Every
workflow title identifies its operator or availability:

- **[User]** — performed in the LumenClip web app with current screenshots.
- **[Proposed user workflow]** — intended app behavior that is not implemented.
- **[MCP]** — callable through the partially implemented MCP server.
- **[Proposed MCP]** — requires tools or resources that are not callable yet.

## Shared rules

1. A manual **Generate** action creates an unpublished, unscheduled draft.
2. Scheduled live automations can auto-publish only when a compatible account
   is selected and the provider supports the output type.
3. Review generated claims, media rights, captions, and platform settings before
   publishing.
4. **Mark as published** records local evidence only. Use **Post to social** to
   send a slideshow through PostFast or **Link published post** to save a live
   URL and provider-native ID.
5. If the UI does not expose a documented action, do not substitute an internal
   endpoint unless the guide explicitly labels it as a developer fallback.
