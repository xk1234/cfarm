## Adding roadmap work

When an initiative moves to **Now**:

1. Add a focused page in this folder.
2. State the intended outcome and observable completion criteria.
3. Link workflow evidence and implementation references rather than copying
   them into the task page.
4. Add the page to `roadmap/meta.json` and link it from the **Now** table.
5. Delete the page after the work ships and fold durable facts into `STATE.md`
   and the appropriate reference guides.
